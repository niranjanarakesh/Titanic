# Titanic
This project performs data cleaning, exploratory data analysis (EDA), feature engineering, and machine learning on the Titanic dataset to analyze passenger survival patterns and predict survival outcomes.


----> Features:

1.Data preprocessing and cleaning
  
2.Feature engineering

3.Exploratory Data Analysis (EDA)

4.Machine Learning

5.Data splitting and preprocessing

6.Decision Tree Classification

7.Model evaluation using accuracy metrics



----> Technologies Used:

Python  |  Pandas |  NumPy | Matplotlib | Scikit-learn 



----> Dataset:

The project uses the Titanic passenger dataset containing information such as:

Passenger Class

Age

Gender

Fare

Embarkation Port

Family Details

Survival Status


----> Objective:

The main objective is to identify factors that influenced passenger survival during the Titanic disaster and build a predictive model capable of estimating survival outcomes based on passenger attributes.


----> Results:

The analysis reveals strong relationships between survival and factors such as:

Passenger class

Gender

Age

Family size

Embarkation port

----> A Decision Tree Classifier was trained to predict passenger survival using these features.
